Project Details

Neutralising Sensationalized Headlines

This project is an NLP-based text transformation tool that converts sensationalized news headlines into neutral and factual tones. The system focuses on detecting exaggerations, rewriting sentences to remove bias, and maintaining the original meaning of the headline. It’s useful for news agencies and readers aiming for objective reporting.
Technologies used: Python, Natural Language Toolkit (NLTK), Regular Expressions, TextBlob.

Medication Reminder

Medimate is a user-friendly web application designed to help users manage and track their daily medication schedules efficiently. Users can set reminders for multiple medications, log doses, and receive timely notifications to ensure adherence to their routines.
Technologies used: HTML, CSS, JavaScript, LocalStorage, optional Bootstrap.

Banking-Term Deposit Prediction

This project is a Machine Learning model that predicts whether a bank client will subscribe to a term deposit based on historical client data. Features include age, balance, job type, and previous subscription history to train the model.
Technologies used: Python, Pandas, Scikit-learn, Jupyter Notebook, Matplotlib/Seaborn.

Sensor-Based Vehicle Space Management

This project is a smart parking management system that detects and displays available parking spaces in real-time. It uses sensors and microcontrollers to monitor occupancy and sends the data to an online dashboard using ThingSpeak. The system helps drivers quickly locate free parking spaces, reduces congestion, and improves overall parking efficiency.
Technologies used: Arduino/Microcontrollers, Ultrasonic/IR Sensors, ThingSpeak IoT platform, C/C++, basic HTML/CSS.
